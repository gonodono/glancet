smoothScrollByOffset

abstract fun smoothScrollByOffset(offset: Int)

Translates to a RemoteViews.setInt call for smoothScrollByOffset.

This seems to be effective only with offset values of magnitude >= 2, though that's not specific to this library. The same behavior can be observed in a classic widget setup as well.

NB: This function has no effect on API levels 30 and below. Glance performs only full updates, and the platform didn't start preserving the relevant state across such updates until API level 31.
